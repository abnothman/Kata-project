Kata : Conversion de Nombre en Chaîne

Ecrire un algorithme qui transforme une nombre (compris entre 0 et 100) en une chaîne de characters. Cet algorithme doit suivre les règles suivantes :
  • Si le nombre est divisible par 3 ou contient 3, ajouter "FOO".
  • Si le nombre est divisible par 5 ou contient 5, ajouter "BAR".
  • Si le nombre contient 7, ajouter "QUIX".
  • Si aucune règle ne s'applique, retourner le nombre sous forme de chaîne.
Exemples :
Input Output
1 "1"
3 "FOOFOO"
5 "BARBAR"
7 "QUIX"
9 "FOO"
51 "FOOBAR"
53 "BARFOO"
33 "FOOFOOFOO"
15 "FOOBARBAR"
